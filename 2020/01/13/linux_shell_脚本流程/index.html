linux_shell_脚本流程

shell变量
shell是没有数据类型的

1.用户自定义变量
    从右到左赋值，
    赋值运算两边不能有空格，
    引用shell比那辆的值一定要加$
    Y=y   X=$Y   echo $X   y
    ubset X //删除X的值，echo $X  为空

2.位置变量(命令行的参数)
    $0    脚本文件名
    $1 ~ $9    命令行的第一个参数 ~ 第九个参数
    ${10}    命令行的第十个参数
3.预定义变量
    $#    命令行参数的个数(不包含脚本文件名)
    $@    所有的命令行参数
    $*    所有的命令行参数
    $?    上一行命令若果结果是0  代表执行成功     为1则失败

4.环境变量
    PATH        标准环境变量
    HOME        查看HOME
    SHELL        查看SHELL
    HISTSIZE    历史命令的个数

    env    export    两个变量都是查看环境变量

    \        转移字符



shell语句
1.说明性语句    以 # 开头   #！/bin/bash
2.功能性语句
    1、read：从标准输入读一行赋给shell变量
    read VAR1  VAR2  VAR3。。。
    echo    $VAR1    $VAR2    $VAR3

    expr 3 + 2 \* 6        算数运算

    2、expr ( + - \* / \( \))    整数算数运算命令
    read A
    expr $A + 2 \* 6
    NUM=`expr 6 + \( 5 \* 5 \)`
    echo $NUM
    echo `expr $NUM \* 6`     
    注意：
        乘号和小括号要加转移符
        运算两边要留有空格
        若果把算数的结果赋给变量要用命令置换
    3、test     测试对象：整数    字符串    文件类型
        整数测试
        a -eq b     判断是否a是否等于b
        a -ne b        判断是否a和b不相等
        a -gt b        判断是否a大于b
        a -lt b        判断是否a小于b
        a -ge b      判断是否a大于等于b
        a -le b        判断是否a小于等于b

        测试字符串
        s1 = s2        a等于b
        s1 != s2    a不等于b
        测试的时候两边要留有空格
        test -z    s1  测试长度
        文件类型
            -    普通文件
            d    文件夹
            l    链接文件
            c    字符设备文件
            b    块设备文件
            p    管道文件
            s    套接字文件

        文件类型测试
            -d file_name 该文件是否是文件夹
            -f file_name 该文件是否是普通文件
            -S file_name 套接字文件
            -L file_name 连接文件
            -c file_name 字符快文件
            -b file_anme 块文件
            -p file_name 管道文件
            -e file_name 文件是否存在
3.结构性语句
        1、条件语句
            #双分支

            if 表达式
            then
                命令表
            else
                xxx
            fi

            #多分支
            if [   ]
            then
                xxx
                xxx
            elif [  ]
            then
                xxx
                xxx
            else
                xxx
                xxx
            if


            #嵌套
            if [  ]
            then
                xxx
                xxx
            else
                if [   ]
                then
                    xxx
                    xxx
                fi
            fi


        2、开关语句
            case     .esca

            模式中可用或(|) 来连续的判断
            例如：
                YES|yes|y|Y)

            case   xxx  in
                模式1)
                    命令表1
                    ;;————————》break
                模式2)
                    命令表2
                    ;;

                ......
                *)——————default
                    命令表
                    ;;
            esac
