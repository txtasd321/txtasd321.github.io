结构体类型

结构体类型

struct date
{
int year;
int month;
int day; //成员变量之间用分号隔开，成员变量之间数据类型可以相同也可以不同
}

struct date 才是结构体数据类型，struct只是一个关键字，date只是结构体名，此时没有分配空间

d1是一个局部变量，d2是一个全局变量

struct date d1 = {2020, 1, 13}
访问结构体成员变量名：
结构体变量名.成员名 d1.year d1.month d1.day

结构体变量之间是可以直接进行赋值的：

1、无名结构体：结构体名

2、结构体指针

3、结构体数组

通过指针访问结构体成员
指针名->成员变量名

4、字节对齐（32os)
系统会一次性分配4个字节
double一次性分配4个字节，分两次分配

5、共同体

==注意==
1.所有成员变量的首地址一致
2.共同体变量的内存空间取决于成员内存最大的成员
3.耨遗失可，只能取出一个变量来

6、判断计算机大小端序
大端序：低地址存高数据
小端序：高地址存高数据

7、枚举

给成员赋值时只能赋值为整形数
一旦成员的值确定，就不能再进行赋值
如果不给第一个初始化，默认值为0，后面的每个值加1
